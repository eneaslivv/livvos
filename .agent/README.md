# Antigravity Agent Configuration

This directory contains configuration for the Antigravity AI agent.

## Skills

The `skills/` directory contains "Antigravity Skills" - structured instructions that extend the agent's capabilities.

- **`cli_ops`**: Bridge to the project's custom CLI (`skills/` in root) for backend/domain tasks.
- **`frontend_engineering`**: Guidelines for React/Tailwind development.
- **`testing_workflow`**: Instructions for using Vitest.

## Workflows

The `workflows/` directory (if present) contains multi-step procedures for complex tasks.
